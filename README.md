# Super Mario Python Platformer (v2.1.4)

A feature-complete, Mario-inspired platformer built from scratch using **Python** and **Pygame**. This version features a dynamic combat system, multiple power-ups, and 20 unique levels of increasing complexity.

## 🚀 Key Features

* **20 Procedural Levels:** Experience a variety of challenges with levels generated through a custom template system.
* **Dynamic Combat:** Beyond jumping on heads, you can perform **Directional Punches** with knockback and **Projectiles** (Fire/Ice).
* **Power-Up System:** Includes six distinct items:
    * ⭐ **Star:** Invincibility and instant enemy defeat.
    * 🔥 **Fire Flower:** Shoot bouncing fireballs.
    * ❄️ **Ice Flower:** Freeze enemies in their tracks.
    * 🍄 **Mushroom/Golden Mushroom:** Enhanced jump height and gravity control.
    * 🪙 **Coins:** Collect 20 to earn an extra life.
* **Smart Entity AI:** Enemies feature health points (HP), damage cooldowns, and ledge-detection to prevent them from falling off platforms.
* **Robust Engine:** * Dynamic camera system that follows the player.
    * Particle effects for breaking bricks and defeating enemies.
    * **Asset Fallback System:** The game automatically generates colored placeholders if assets are missing, making the code runnable out of the box.
